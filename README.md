# Craycast - Your Broweser's Raycast

Hi there 👋
I am Bhavit Grover, a keen developer who wants to expand his skills. I had recently stumbled upon a piece of software called Raycast, and was much inspired by it to create a Browser version of it.

## Description
Being a developer, I am awarae of this annoying feeling of **navigating through hundreds of opened tabs**. To solve some of that issue, I built **Craycast**. Craycast stands for Chrome + Raycast, basically this extension features some of the utitilty that Raycast provides on the desktop application. It is something, that anyone can download, and use it in their daily lives. You just need to press **Ctrl + Shift + K** in order to open the menu and get started.

## Features

1) Craycast features the ability to **navigate through tabs**, and **saved bookmarks** with a **lightning fast search menu**, just by pressing ```Ctrl + Shift + K```
2) Pressing ```Tab``` also reveals an **AI Mode** where you initialize it first, using a **Hack Club API key**, which you can find [here](http://ai.hackclub.com/).

## Installation Instructions

1) Clone this Github repository.
2) Open it in an editor or IDE.
3) Write the following commands in the terminal
   
   ```
   npm install
   ```
   ```
   npm run build
   ```
5) You would now see a ```.output``` folder appear in the file system.

### NOTE
You can just skip the steps above and download the project from [here](https://drive.google.com/drive/folders/1nxj86SX_aElGFZ_GLCkcDKoSxfpnoDAE?usp=sharing).

7) Open Chrome browser, go to ```chrome://extensions``` and over there on the top right menu, enable Developer Mode.
8) Then you can click on Load Unpacked Extension, and select the ```chrome-mv3``` folder present in the ```.output``` directory.
9) You can now open a website, and press ```Ctrl + Shift + K``` to open Craycast.

### NOTE
Chrome blocks the usage of such shortcuts and extensions on the settings, and the new tab page. It is therefore recommended to open either ```google.com``` or another site in order to view the Craycast window.
